# 编程练习：哈夫曼编码的编码与解码

## 功能

1. 实现对任意文件的进行哈夫曼编码
2. 统计文件中不同字符的种类和出现频率
3. 构建相应的哈夫曼树，并给出各个字符的哈夫曼编码
4. 对源文件进行哈夫曼压缩编码形成新的压缩后文件
5. 编写解压缩文件对压缩后文件继续解码还原成源文件

## 效率

| 文件 | 说明 | 压缩比 |
| --- | --- | --- |
| Sample.txt | 包含大量重复字符的纯文本 | (38KB / 47KB) 80.85% |
| Sample.html | 包含中文字符的网页 | (88KB / 95KB) 92.63% |
| Sample.png | 图像 | (2238KB / 2518KB) 112.39% |

## 需求

- Visual Studio 2017
